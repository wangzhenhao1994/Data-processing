This is a python program to solve 1D Schroedinger equation.

1. How to run?
-------------------------
	- In terminal write: python schroedinger.py
	
2. Used method:
-------------------------
	1. Numerov Algorithm: for boundary value.
	2. Simpson Method: for normalization.
	3. Secant Method: for searching eigenvalue.

For solving SE for Hydrogen atom with total angular momentum quantum number
in the potential one can use hydrogen_schroedinger.py as

3. To solve Hydrogen atom's SE
    - In terminal write: python hydrogen_schroedinger.py
